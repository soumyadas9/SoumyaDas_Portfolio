AI Study & Task Planner

Link to Project: https://ai-studyplanner-e5cd.onrender.com/

• Built a full-stack productivity web app using Flask, Python, HTML/CSS, and SQLite that generates intelligent daily study schedules.

• Developed an AI-driven scheduling engine that allocates study blocks based on task difficulty, deadlines, remaining workload, and available daily hours.

• Implemented a realistic human-focus model with dynamic work sessions (1.5h, 1h, 0.5h) and adaptive break intervals for sustainable productivity.

• Created an efficiency tracking & feedback system that compares planned vs completed work, calculates productivity, and delivers personalized insights.

• Designed a habit-building streak system and performance analytics to improve consistency and highlight weak focus areas using historical study data.
